# Avito AI Assistant

AI-ассистент для улучшения объявлений на Авито. Помогает продавцам создавать качественные описания и определять рыночную цену.

## 🚀 Технологии

- **React 18** + **TypeScript**
- **Vite** — сборка
- **React Router v6** — маршрутизация
- **Zustand** — управление состоянием
- **TanStack Query** — работа с серверными данными
- **Material UI** — компоненты интерфейса
- **Ollama / Grok API** — интеграция с LLM

## 📦 Установка и запуск

### Требования
